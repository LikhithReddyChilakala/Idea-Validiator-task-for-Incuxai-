# Idea-Validiator-task-for-Incuxai-
The "Idea Validator" Micro-Agent 

## The Problem:
Students have great project or startup ideas but lack access to  immediate, constructive, and realistic feedback to validate them before spending time building.

## The Brief: 
Create a single-purpose AI chat interface where users submit a project or startup idea, and an 
AI agent provides a realistic, 3-point constructive critique. You must connect to a free LLM API 
and configure the model with a highly specific "persona" (e.g., "Harsh but fair Silicon Va ley 
Investor") via system instructions. The UI must cleanly distinguish between user messages and 
the AI agent's responses.
